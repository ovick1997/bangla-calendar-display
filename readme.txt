== Description ==

Bangla Calendar Display is a simple plugin that shows the current Bengali date and time on your WordPress site. It can be used in posts or pages using the `[bangla_calendar]` shortcode. The plugin calculates the Bangla date based on the Gregorian calendar and handles leap years.

== Installation ==

1. Upload the `bangla-calendar-display` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[bangla_calendar]` shortcode in any post or page to display the Bangla calendar.

== Frequently Asked Questions ==

= How do I use the Bangla Calendar? =

Simply add the shortcode `[bangla_calendar]` to any post or page where you want to display the Bangla date.
